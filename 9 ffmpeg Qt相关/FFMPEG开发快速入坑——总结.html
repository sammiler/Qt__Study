至此，整个FFMPEG开发快速入门基础都已经全部讲解完成了。我们对所有使用到的API做个总结整理：


一、媒体文件分流解析处理

这些API都属于 libavformat 库中的函数

  1. avformat_open_input() / avformat_close_input();
  2. avformat_seek_file()
  3. av_read_frame()


二、音视频解码器

这些API都属于 libavformat 库中的函数

  1. avcodec_alloc_context3() / avcodec_free_context()
  2. avcodec_parameters_to_context()
  3. avcodec_open2() / avcodec_close()
  4. avcodec_send_packet() / avcodec_receive_frame(): 这两个是真正解码处理时使用的函数，详细在下一章节讲解

重点是注意： avcodec_send_packet() 送入数据包 和 avcodec_receive_frame() 获取解码后的帧不同步


三、视频格式转换

这些API都属于 libswscale库

  1. sws_getContext() 根据要输入输出图像的 宽高和 像素格式 创建转换器
  2. sws_scale() 根据输入图像数据进行实际的转换操作，结果输出到输出缓冲区上
  3. sws_freeContext()释放转换器

重点是不同视频图像格式的内存布局


四、音频格式转换

这些API都属于 libswresample库

  1. swr_alloc() / swr_free()：创建/释放音频转换器
  2. swr_init()、swr_alloc_set_opts()：设置转换参数
  3. swr_convert(): 进行实际的音频数据转换

重点是不同采样格式、不同通道情况下，音频数据的内存布局


五、音视频编码处理

这些API都属于 libavformat 库中的函数

  1. avcodec_find_encoder() / avcodec_find_encoder_by_name();
  2. avcodec_alloc_context3() / avcodec_free_context()
  3. avcodec_open2() / avcodec_close()
  4. avcodec_send_frame() / avcodec_receive_packet()

重点是注意：

  1. avcodec_send_frame() 送入帧 和 avcodec_receive_packet() 获取数据包不同步
  2. 对于音频编码器，不同的编码格式，其采样格式、每次送入样本数量有固定要求


六、音视频混流处理

这些API都属于 libavformat 库中的函数

  1. 1. avformat_alloc_output_context2() / avformat_free_context()
  2. 2. avformat_new_stream() 和 avcodec_parameters_from_context()
  3. 3. avio_open() / avio_closep()
  4. 4. avformat_write_header() / av_write_trailer()
  5. 5. av_interleaved_write_frame()
