本文介绍了QML中的Rectangle元素，包括如何创建简单矩形、渐变矩形、带边框的矩形、圆角矩形以及如何实现圆形和圆环。Rectangle元素提供了color、gradient、border和radius等属性，允许开发者灵活定制矩形的外观和样式。

在 QML 中，有许多可用的元素，其中最常见的一个是 Rectangle，可以用它来创建矩形形状的图形。

Rectangle 继承自 Item，并提供了 color 和 gradient 属性，以使用纯色或者渐变色填充矩形区域。此外，还可以通过 border（border.width、border.color）定义矩形边框的宽度和颜色，并利用 radius 创建圆角矩形。

完整的思维导图如下：

简单的矩形

使用 Rectangle 创建矩形非常简单，以下是一个基本的示例：

import QtQuick

Rectangle {
            
    width: 100
    height: 100
    color: "green"
}

在这个例子中，我们创建了一个大小为 100x100 的绿色矩形。可以看到，Rectangle 元素有三个属性：width、height 和 color，其中 width 和 height 定义了矩形的宽度和高度，而 color 定义了矩形的颜色。

渐变的矩形

倘若我们不想使用纯色填充矩形，而是想设置一个渐变色，则可以通过 gradient 属性来实现。

Rectangle {
            
    width: 100
    height: 100
    gradient: Gradient {
            
        GradientStop {
             position: 0.0; color: "lightgreen" }
        GradientStop {
             position: 1.0; color: "green" }
    }
}

渐变是由两种以上颜色定义的（会无缝融合），在这里被指定为一组 GradientStop 子项，其中每个子项对应了 0.0 ~ 1.0 梯度上的一个位置和颜色，分别由 position 和 color 属性来表示。

这里，我们为位置 0.0 处指定了颜色 lightgreen，为位置 1.0 处指定了颜色 green，因此矩形会在垂直方向上由淡绿色向绿色渐变（若想在水平方向上渐变，可以利用 rotation 属性顺时针旋转）。

注意： 若同时指定了 gradient 和 color，则将会使用 gradient 而忽略 color。

带边框的矩形

此外，我们还可以添加其他属性来进一步定制 Rectangle。例如，通过 border 定义矩形的边框：

Rectangle {
            
    width: 100
    height: 100
    color: "green"
    border.color: "black"
    border.width: 2
}

由于 border 是一个分组属性（拥有子属性 color 和 width），因此可以使用点表示法或者组表示法定义。

上面使用的是点表示法，border.color 和 border.width 将边框色设置为了黑色并将边框宽度设置为 2。对应的组表示法为：border { color: "black"; width: 2 }

圆角矩形

通常，圆角矩形也很常见，我们可以使用 radius 属性来定义矩形的圆角半径：

Rectangle {
            
    width: 100
    height: 100
    color: "green"
    radius: 10
}

这里，我们将 radius 设置为 10，这会创建一个具有 10 像素圆角的矩形。

圆形 & 圆环

扩展一下，实现一个简单的圆形：

Rectangle {
            
    width: 100
    height: width
    color: "green"
    radius: width/2
}


对于圆形，其特点是宽高相等，且圆角半径是宽度/高度的一半。因此，这里我们使用了属性绑定，将 height 的值绑定为 width，并将 radius 的值绑定为 width/2，这样就可以达到动态更新的目的了。

如果想实现一个圆环呢？也很容易！

Rectangle {
            
    width: 100
    height: width
    border.color: "green"
    border.width: 20
    radius: width/2
}


想象一下，圆环和圆形的差异点是什么？圆环没有填充圆形区域，而是拥有一个固定宽度的边框。因此，这里我们并没有设置 Rectangle 的 color，而是利用 border 将边框色设置为了绿色，边框宽度设置为了 20。

总之，Rectangle 是 QML 中一个非常有用的元素，可用于创建各种各样的形状。通过调整其属性，我们可以轻松地定制矩形的外观，使其满足最终的需求。
